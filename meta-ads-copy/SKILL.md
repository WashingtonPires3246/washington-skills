# Meta Ads Copy

## Contexto do stack Meta
- Campanhas com Advantage+ Audience (segmentação ampla)
- Criativo é a principal alavanca de performance
- Objetivo predominante: conversão (venda direta ou lead)

## Como o Advantage+ muda a copy
Com segmentação ampla, a copy precisa filtrar a audiência por si mesma:
- Mencionar explicitamente para quem é
- Criar identificação imediata com a dor
- Usar ganchos que repelem quem não é o público

## Estrutura de anúncio de conversão
GANCHO → PROBLEMA → SOLUÇÃO → PROVA → CTA

## Tipos de gancho
- Dor: "Você já tentou de tudo para emagrecer e nada funciona?"
- Identificação: "Se você tem mais de 35 anos e sente que seu metabolismo travou..."
- Revelação: "O que a maioria dos nutricionistas não te conta..."
- Prova social: "174 mil seguidoras já conhecem esse protocolo."
- Urgência suave: "Você viu esse conteúdo semana passada. Ainda não decidiu?"

## Output padrão
1. 3 variações de Primary Text
2. 3 Headlines
3. 2 Descrições
4. Sugestão de formato de criativo
5. Temperatura indicada (cold/warm/hot)
